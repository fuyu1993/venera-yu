一款支持阅读本地和网络漫画的漫画阅读器。

## 功能特性
- 阅读本地漫画
- 使用 JavaScript 创建漫画源
- 阅读网络漫画源
- 管理收藏漫画
- 下载漫画
- 查看评论、标签及其他漫画信息（需漫画源支持）
- 登录进行评论、评分等操作（需漫画源支持）

## 从源码构建
1. 克隆仓库
2. 安装 Flutter，请参阅 [flutter.dev](https://flutter.dev/docs/get-started/install)
3. 安装 Rust，请参阅 [rustup.rs](https://rustup.rs/)
4. 为你的平台构建：例如 `flutter build apk`
